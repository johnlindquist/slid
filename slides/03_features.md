# Features

## Big Text Headers

Every `# Header` becomes ASCII art via `ink-big-text`.

## Gradient Colors

Headers use the `cristal` gradient for visual appeal.

## Markdown Support

Full markdown rendering with:
- **Bold** and *italic* text
- `inline code`
- Code blocks
- Lists (like this one!)
- And more...

## Scrollable Content

Long slides scroll smoothly:
- ↑/↓ arrows for line-by-line
- Space for page down

## Recording Playback

Drop `.cast` files in the `slides/` folder.
They become playable demo slides!

## File-Based Slides

```
slides/
├── 01_intro.md
├── 02_demo.cast
├── 03_conclusion.md
```

Numbered prefixes control order.
