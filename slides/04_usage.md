# Usage

## Running the App

```bash
bun run index.tsx
```

## Creating Slides

1. Create markdown files in `slides/`
2. Use numbered prefixes: `01_`, `02_`, etc.
3. First `# Header` becomes the big title

## Recording Demos

```bash
# Record a demo
asciinema rec slides/05_demo.cast

# Do your thing...
# Type 'exit' when done
```

## Pro Tips

1. **Increase font size** before presenting (Cmd +)
2. **Full screen** your terminal
3. **Practice** the flow beforehand
4. Keep demos **short** (under 2 minutes)

## That's It!

Simple, reliable, hacker-friendly.

Press **q** to quit.
